# MSc-Stroke-Prediction-Project
Final data science project for the completion of the MSc - Data Science program offered by the University of Nicosia.
This repository contains the following files related to the project:

1. Project Code (part 1): The initial cleaning and dataset preparation.
2. Project Code (part 2): The following EDA-exploration phase where the data are also prepared for the Machine Learing phase.
3. Project Code (part 3): The ML model experimentation and evaluation phase.
4. Project Report: The final report of all the work and results (interim-paper style prepared)
5. Project Presentation: The powerpoint presentation I used for the final examination and report (in pdf).

More details about the data, structure, procedures and results are contained in the files themselves (both code files and report files).
